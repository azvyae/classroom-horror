# Classroom Horror Classical Web Game

Escape the school by looking for the Green door before you are caught by the lurking monster.

Try to hide in one of the classrooms, marked with the red doors.

## Controls

- **W** to move forward
- **A** to move the camera left
- **S** to move backward
- **D** to move the camera right
- **E** to interact with objects
- **M** to toggle the minimap (not works on Hardcore mode)
